新生儿各种手续

记录新生儿的各种手续。

不同居委会/社区等对材料的要求不同，建议先联系后再针对性准备。

出生证明

现在出生证明可以线上办理，下载北京通，首屏出生证申领。尽量在小孩儿出生一周后，满月前办理。

协和新生儿建档 + 复查

新生儿出生后戴的手环上有 ID 号，产妇和新生儿出院时，协和也会告知 ID 号。家长自行到收费处提供新生儿 ID 直接建档。42 天复查时，提前 7 天在协和 App 上建档，然后预约儿科新生儿或儿童保健的号。

户口

需要的材料：

  1. 夫妻双方身份证。
  2. 夫妻双方户口本。
  3. 夫妻双方结婚证。
  4. 夫妻双方房本。
  5. 出生证明原件。

体检疫苗新生儿建本

需要的材料：

  1. 出生证明原件。
  2. 出生证明复印件。
  3. 北京市预防接种证(出生接种疫苗时，出生医院办理)。
  4. 出生时卡介苗、乙肝疫苗第一针接种记录，儿童转诊单。我们的这个都在预防接种证上。
  5. 北京市母子健康手册儿童保健记录(怀孕后到社区卫生服务中心领取)，用于记录体检结果。

预约体检和打疫苗都是在“儿宝宝”小程序里预约，要先到社区卫生服务中心预防保健科建本才能预约，疫苗提前一个月放号，体检提前一周放号。

自费疫苗

参考了知乎这个回答，必选的自费疫苗包括水痘疫苗、五联疫苗、13 价肺炎疫苗、肠道病毒 71 型灭活疫苗(手足口疫苗)、五价轮状病毒疫苗，有可能的话免费的流脑疫苗换成自费的流脑疫苗，注意是流脑疫苗，不是乙脑疫苗，乙脑疫苗选免费的减毒疫苗就可以。

银行卡

银行卡主要是医保缴费使用，北京银行需要的材料包括：

  1. 宝宝户口对应的户口本原件，即使和代办人不在一个户口本也没关系。
  2. 出生证明原件。
  3. 代办人本人身份证。

医保卡

  1. 出生证明原件 + 两份复印件。
  2. 户口本原件，首页 + 宝宝本人页复印在同一张 A4 纸上两份，宝宝本人页再单独复印两份。
  3. 填表时要求选三家医保机构，需要提前准备，不过孩子看病一般都去 A 类医院或者专科医院，意义不大，可以填一个家旁边的医院方便开药。
