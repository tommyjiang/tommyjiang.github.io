Data VS Performance

Revisiting Unreasonable Effectiveness of Data in Deep Learning Era

核心结论：数据增长与模型性能增长基本呈对数线性关系，即数据增加一个数量级，模型性能提高的点数相同。从 ImageNet 1M 数据集扩展到 JFT-300M 数据集，4 epoch 需要在 50 块 K-80 GPU 上训练 2 个月。

Object Detection

Framework

RefineDet

CVPR 2018 paper，检测网络结构，主体包括 ARM(Anchor Refinement Module) 和 ODM(Object Detection Module)，利用 TCB(Transfer Connection Block)连接两者，将 ARM 的 feature 传给 ODM。

ARM 采用 VGG/ResNet，预测分类和回归时采用 3*3 的滑动窗口，从 stride 8、16、32、64 的 feature map 分别设置 scale 为 4 倍 stride，ratio 为 0.5/1/2 的 anchor。ODM 类似 SSD，TCB 借鉴 FPN 思想，融合高层 feature map 信息。训练和测试时，ARM 都会舍弃分数低于阈值的 anchor box。

结果：RefineDet320，VGG 16，40.3 FPS。RefineDet512+，ResNet-101，41.8 mAP on COCO。

YOLO-LITE

关键指标：

  • 定位：无 GPU 设备
  • 个人电脑 21 FPS，网站 7 FPS
  • 7 层，482 MFLOPS 计算量
  • mAP: PASCAL VOC 33.81%，COCO 12.26%

数据集数量：

  • PASCAL VOC 2007+2012 训练集：5011(20 类)
  • COCO 2014 训练集：40775(80 类**

其他小网络对比：

  • Tiny-YOLOv2 @ 23.7% mAP, 2.4 FPS
  • SSD Mobilenet V1 @ 21.0% mAP, 5.8 FPS
  • YOLO-LITE @ 12.3% mAP, 21 FPS

Acquisition of Localization Confidence for Accurate Object Detection

  • Trick

    Decoupled Classification Refinement(DCR)

    ECCV 2018 paper，思想和 alignment 网络很像，单独训练一个网络用于 Detection 网络结果的分类，训练时采用 Hard Positive 杀误检，测试时所有结果均会经过该 refinement 网络。为减少计算量，提出了采用 RoIpooling 之后 feature 作为网络输入的方法。

Layer

Convolution variation

Deformable Convolutional Networks

ICCV 2017 paper，学习卷积和 RoI pooling 时，点坐标的 offset，解决卷积和 RoI pooling 无法自适应地根据物体调整位置的问题。

  • 卷积层：与普通卷积并列，3*3 卷积，2N channel 的 feature map(N 为卷积核)，输出 feature map 上 N 个点的 x/y offset。
  • RoI pooling：FC 层，输出归一化的 offset，需要乘 feature map 尺寸和 gamma。
  • PS RoI pooling：全卷积，输出 2(C+1) k^2 channel 的 feature map，即对每一类和每一个 position，都输出一个 offset。

结果：COCO，R-FCN mAP 34.5 -> DCN mAP 37.1

Deformable ConvNets v2

DCN v1 提出两个层：Deformable Convolution 和 Deformable RoIpooling。

本文改进：

  • 网络中更多层使用 Deformable Convolution。
  • Deformable Convolution 中加入学习权重。

Pooling variation

Learning Region Features for Object Detection

ECCV 2018 paper，学习 pooling 层的参数。

  • RoI pooling
    • Support Region：RoI
    • Weight：1/R_{bk}
  • Aligned RoI pooling
    • Support Region：RoI
    • Weight：双线性插值，只和临近的四个 pixel 有关
  • Deformable RoI pooling
    • Support Region：整张图
    • Weight：双线性插值

图片坐标 p 编码为 2C 维，RoI 编码为 4C 维，之后做线性变换，变换矩阵可学习。图像特征通过 1*1 的卷积计算。图片坐标如果全部计算，计算量太大，因此采用稀疏采样的方法，每 \sqrt(H * W / 196) 个 pixel 进行一次采样。

结果：ResNet 50 Faster RCNN，COCO，mAP 34.1

Tricks

Drop

DropBlock

新的正则化方法，DropBlock。

文中指出，Dropout 适合 FC 层，但不适合卷积层，因为卷积层的 feature 是空间相关的。

参数包括两个，block_size 和 gamma，gamma 是用 keep probability、feat size、block size 计算得到的。训练开始时，keep prob 取 1，然后逐渐减小到 0.9。

结果：ImageNet 上，ResNet 50，top 1 76.51 -> 78.35，COCO 上，RetinaNet，mAP 36.5 -> 38.2。
