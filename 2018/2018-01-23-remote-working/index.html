服务器远程调试工具

工欲善其事，必先利其器。

远程调试 PyCharm Professional

写代码简单，调代码难，在同一套环境中调代码更难，配过环境的同志都懂得环境配置的痛苦。直接在服务器上调试代码可以完美解决这个问题，只需稳定的网络就能保证环境的一致性。PyCharm Professional可以远程ssh到服务器调试，具体流程可以参考使用 Pycharm 通过 SSH 同步代码与远程调试。

远程文件传输 GUI Cyberduck

有时候需要把服务器的文件拷贝到本地，虽说可以用scp命令但没有 GUI 总感觉不太直观。Mac 下可以用免费的Cyberduck建立 SFTP 连接，非常方便。
